# Netherlands work permit / residence application pack

Date: 2026-06-17

## Immediate conclusion

For a Netherlands employment route such as **Highly Skilled Migrant (Kennismigrant)**, the IND states that **only an employer recognised by the IND can apply** for the permit. The candidate cannot independently submit this application as if it were a personal licence.

Most likely route for Deloitte / similar employer: **Highly Skilled Migrant residence permit**, if the Dutch employer is an IND-recognised sponsor and salary meets the threshold.

Official IND page checked: https://ind.nl/en/residence-permits/work/highly-skilled-migrant

## 2026 IND salary thresholds checked

Source: https://ind.nl/en/required-amounts-income-requirements

For first application / extension / change of employer in 2026:

- Highly skilled migrant reduced salary criterion: **€3,122 gross/month**
- Highly skilled migrant younger than 30: **€4,357 gross/month**
- Highly skilled migrant 30 or older: **€5,942 gross/month**
- European Blue Card: **€5,942 gross/month**
- Reduced salary criterion European Blue Card: **€4,754 gross/month**

Important: holiday allowance is not included in the IND salary criterion. Only fixed monthly amounts meeting IND rules can count.

## What must be confirmed before any submission

1. Employer / sponsor
   - Exact Dutch employing entity name.
   - Confirm it is an IND recognised sponsor.
   - Confirm whether Deloitte itself or a payroll/umbrella entity will be sponsor.

2. Job offer details
   - Signed or draft employment contract.
   - Start date.
   - Role title.
   - Gross monthly salary excluding holiday allowance.
   - Whether salary is market-conform.

3. Candidate details
   - Current nationality / passport country.
   - Current country of residence.
   - Whether MVV entry visa is required.
   - Whether candidate already has Dutch/EU residence status.
   - Age at application date.
   - Whether reduced salary criterion might apply via recent graduation/orientation-year eligibility.

4. Family members, if relevant
   - Whether spouse/partner and child apply together.
   - Passports and relationship/birth documents.
   - Translations/legalisation/apostille status if required.

## Documents to prepare from candidate side

### Core
- Passport scan: personal details page, full validity visible.
- Updated CV.
- Signed employment contract once available.
- Completed antecedents declaration if requested by sponsor/IND.
- Passport photo / biometrics later via IND appointment.

### If family applies together
- Spouse/partner passport.
- Child passport.
- Marriage certificate / partnership proof as applicable.
- Child birth certificate.
- Legalised/apostilled and translated versions if not already accepted by Dutch authorities.

### If relying on reduced salary criterion
- Diploma / graduation proof.
- Graduation date proof.
- Any orientation-year eligibility evidence.

## What the employer must do

- File the application through IND Business Portal or written form.
- Pay the IND application fee.
- Submit sponsor/employment documents.
- Notify candidate after decision.
- Provide confirmation of work authorization start date if positive decision arrives before residence card is ready.
