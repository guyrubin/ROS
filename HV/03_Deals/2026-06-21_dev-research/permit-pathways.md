# Permit & Zoning Pathways — 5 Dev Candidates (Antwerp BE + Den Haag NL)

## Executive takeaway

- **Permit is on the critical path for all five.** Four of five depend on a *use/unit-creating* permit (splitsing/woningvorming NL, opdeling/regularisatie BE) — i.e. the deal thesis is the permit, not just a build approval.
- **The two binding constraints:** (1) **Den Haag woningsplitsing is GATED until 1 Apr 2026** — no split permit thesis is bankable before that date and the post-April verordening must be re-read per wijk; (2) **BE opdeling is capped by the Antwerp Bouwcode 100 m²/15 m² rule** — it limits how many units you can carve from a house with outdoor space, and can kill an aggressive unit count.
- **Heritage is a cost/timeline multiplier on 4 of 5** (Het Zuid, historic centre, Duinoord beschermd stadsgezicht). It rarely kills, but it pushes toward the longer procedure, welstand/Onroerend-Erfgoed review, and a higher build spec.
- **No outright Red.** Verdicts: **2 Green, 3 Conditional.** The Conditionals all hinge on a single verifiable fact (zoning use, opdeling cap, or the 1-Apr-2026 split rules).

## Critical-path placement (per candidate)

**Permit sits BEFORE build and, on the use-changing deals, gates the whole thesis. Sequence the front end explicitly:**

- **A1 (Zuidpark):** Critical path = `stedenbouwkundig uittreksel + Onroerend Erfgoed status check → architect concept reconciled with heritage → regular omgevingsvergunning (incl. 30-day public inquiry) → appeal window → build`. **~5–7 mo of permit before a spade.** Heritage advice on the park-front elevation is the long pole. If a multi-unit split is intended, run the opdeling feasibility (100 m²/15 m² rule) at concept stage — do not assume unit count.

- **A2 (Oever 18):** Critical path = `confirm 3rd unit is regularisable (agent + notary stedenbouwkundige inlichtingen) → regularisatie + reno omgevingsvergunning → heritage advice → build`. The **regularisation is the binding node** — until it is granted the asset is a legal 2-unit, and the underwriting must survive that downside. Place the regularisation decision as a go/no-go milestone, not an assumption.

- **A3 (Dries 20):** Critical path = `uittreksel + Bouwcode opdeling check → architect re-plan to the cap → omgevingsvergunning (simplified if no inquiry/structure) → build`. **Permit is fast (~3–5 mo)** and **EPC C removes the renovatieverplichting node entirely** — this is the lowest permit-risk BE candidate. The only real gate is how many sellable units survive the 100 m²/15 m² cap + the retail→wonen functiewijziging.

- **D1 (1e Sweelinckstraat 6):** Critical path = `omgevingsplan + monument-register check → confirm post-1-Apr-2026 splitsing eligibility for Duinoord (≥35 m²/unit, Leefbaarometer, parking <90%) → souterrain structural/geohydrology study → uitgebreide omgevingsvergunning + woningvorming vergunning + (poss.) monumentenvergunning → welstand iterations + participation → objection window → build`. **This is the longest and most fragile front end (~6–10 mo)** with two independent kill-switches (souterrain feasibility + the split gate). Underwrite the as-is single-dwelling fallback. Also note: **above the €1.5M cap** — permit risk compounds an already-stretched price.

- **D2 (Stevinstraat archetype):** Critical path = `omgevingsplan + beschermd-gezicht check → reguliere omgevingsvergunning for the reno → (welstand) → build`. **Shortest path (~2–4 mo)** *as long as the thesis stays a single-dwelling deep renovation*. The critical-path discipline here is scope: keep it out of split/optopping territory unless/until you accept the D1-class gate.

## Specialists required (named, where path is non-trivial)

- **BE all three:** local **notary** (stedenbouwkundige inlichtingen, registration vs VAT structuring) + **architect** (obligatory for dwelling-creating/structural works, files via Omgevingsloket). **A1 & A2 additionally:** **Onroerend Erfgoed Antwerpen** consult (heritage scope) + heritage-experienced architect.
- **A2:** a **vergunningenspecialist / advocaat omgevingsrecht** to validate the regularisation route is real (formalisable) and not a hard zoning breach — this is the deal's binding node.
- **D1:** **constructie/geotechnical engineer** (souterrain dig — water table, underpinning) + **Den Haag woningvorming/splitsing specialist** (post-1-Apr-2026 verordening) + **architect with beschermd-gezicht/monument track record** for welstand. Add a **BBL/Bouwbesluit** check on fire/sound separation for any added unit.
- **D2:** architect for welstand if a gezicht overlay confirms; otherwise standard reno permit — lightest specialist load.

## Cross-cutting flags for hv-orchestrator

1. **D1's split upside is not bankable today** — the 1-Apr-2026 Den Haag gate is a hard date. If the unit-add is core to the return, this is a **timing/binding-constraint escalation**: either wait for the published post-April verordening + confirm Duinoord eligibility, or underwrite D1 on reconfig + souterrain m²-add alone (single dwelling), which souterrain risk then dominates.
2. **A3 is the cleanest permit story** (EPC C kills the renovatieverplichting node; fast procedure; central) — the constraint is commercial (opdeling cap on unit count), not regulatory feasibility.
3. **Heritage is verify-per-address everywhere except A3/D2** — none of the heritage flags above are confirmed monument status; they are zone-level sensitivities. A single monument-register hit on A1/A2/D1 lengthens the path and raises build spec but does not by itself kill.
4. **BE renovatieverplichture only bites if EPC is E/F.** A1's and A2's EPC labels are unconfirmed — if E/F, fold the label-D-in-6-yr obligation into scope (aligns with a deep-reno thesis anyway). A3 (C) and the EPC-C centre options are clear.
