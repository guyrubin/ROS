## Correction from user

The search should **not** focus on all of Amsterdam Noord or rural Landelijk Noord. The map provided points to the **prime urban/waterfront Noord zone**:

- Overhoeks
- Schegpark / Tolhuistuin / A’DAM Lookout
- Sausalitolaan / Badhuiskade
- Grasweg
- Docklandsweg / Ranonkelkade edge
- Buiksloterham directly north/west of the canal
- Nearby Hamerkwartier / Asterweg only as secondary extension
- NDSM/Klaprozenbuurt only as secondary extension, not the core target

This is the zone with the best combination of:
- Centrum ferry proximity
- IJ waterfront value
- high-quality newbuild stock
- urban amenities / restaurants / hotel / cultural anchors
- Overhoeks prestige
- Buiksloterham transformation upside
- scarce family/live-work/freehold/self-build style assets

## Area hierarchy

### A-zone: Prime / should be first screen

1. **Overhoeks / Badhuiskade / Sausalitolaan**
   - Best prestige / liquidity / international-buyer depth.
   - Mostly apartments, limited development upside.
   - Buy only if pricing is attractive or unit is unusually large/unique.

2. **Grasweg / Schegpark / Tolhuistuin edge**
   - Very strong lifestyle location: near ferry, IJ, Tolhuistuin, A’DAM Lookout.
   - Strong for newbuild delivered stock and some commercial/live-work product.
   - Development upside comes from commercial floors, waterfront scarcity, and remaining transformation context.

3. **Core Buiksloterham: Monnikskapstraat / Papaverweg / Distelweg / Leen Jongewaardkade / Ridderspoorweg / Klaprozenweg west edge**
   - Best HV hunting ground for actual optionality.
   - More mixed-use, self-build/free-plot history, atelier/live-work, split potential, and newbuild under/just completed.
   - Stronger upside than pure Overhoeks apartments.

### B-zone: Secondary extension

4. **Asterweg / Hamerkwartier**
   - Still close and improving, but more commercial/office and area-transition risk.
   - Interesting for lower €/m² commercial or mixed-use conversion optionality.

5. **NDSM / Klaprozenbuurt**
   - Good long-term transformation story.
   - Further from the exact map focus; include only if pricing or optionality is compelling.

### Excluded from this focused mandate

- Landelijk Noord / Ransdorp / Holysloot / Durgerdam unless the user specifically wants a rural self-build or estate play.
- Generic Amsterdam Noord apartments without waterfront/Overhoeks/Buiksloterham relevance.

## Corrected buy-box for this search

Search priority should be:

1. **Core Buiksloterham live-work/self-build products**
   - Monnikskapstraat, Papaverweg, Distelweg, Ridderspoorweg, Leen Jongewaardkade.

   - Target signals: underpriced newbuild resale, large terrace, parking, IJ view, commercial space that can be controlled or leased well.

   - Target signals: <€8,500/m² for premium product, large terrace, parking, high floor/IJ view.

   - Grasweg, Monnikskapstraat, Ridderspoorweg, Chrysantenstraat, Asterweg.
   - Need clear business plan: own-use, tenanting, creative/studio/health/sport/office, or long-option transformation.
