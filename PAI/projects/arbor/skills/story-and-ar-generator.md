# Story And AR Generator Skill

## Use When

The user wants personalized stories, child-facing explanations, AR concepts, emotional learning games, or creative interventions.

## Objective

Create story-based tools that help children understand emotions, routines, courage, kindness, language, and behavior.

## Output

```md
## Goal

## Child Profile Inputs

## Story Concept

## Characters

## Plot

## Parent Interaction Points

## AR / App Interaction

## Learning Outcome
```

## Rules

* Age-appropriate.
* Warm and imaginative.
* No manipulation or fear-based content.
* Always include parent-child interaction.
