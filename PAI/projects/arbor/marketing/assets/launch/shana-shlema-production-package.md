### VO SCRIPT — HEBREW (shots 10–12)

Real parent speaks this or HeyGen reads it. One continuous take — do not splice between lines. Pause 0.5s between each line as marked.

```
לא ידעתי שזה מצטבר.
[pause]
שמישהו שומר.
[pause]
עכשיו אני רואה שנה שלמה שלה —
[pause]
וזה שלי.
```

**VO director note:** The rhythm of this script is deliberate pausing — not emotional acceleration. Instruct the talent or HeyGen voice model to treat each line as a separate thought that arrives, not a sentence to be read through. The pauses are the meaning. Total VO duration: approximately 8–10 seconds, fitting shots 10–11.

**HeyGen build note (if using HeyGen):** Select the calm HE female voice model. No avatar — VO-only mode over the visual. Set pace to slow (not the default). The script must be entered in the HeyGen interface exactly as written above, including line breaks (which the model interprets as pause markers). Review the audio before cutting — if the pauses collapse, break the script into four separate text inputs and stitch in CapCut.
