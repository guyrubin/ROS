Parent intake

What’s happening?

My 4-year-old melts down every morning before daycare.
Started: three weeks ago · Frequency: weekdays
Tried: rushing less, rewards, consequences
Low
Medium
High
Urgent
Arbor response format

Read

This looks like a repeated transition problem, not a character problem. The pattern matters more than one morning.

Do Today

  • Reduce choices to two simple options.
  • Prepare the first step the night before.
  • Name the transition: “Now we move from home mode to daycare mode.”
  • Track trigger, duration, and recovery.

Escalate If

If distress worsens, appears across settings, or includes harm to self/others, prepare a professional handoff note.
