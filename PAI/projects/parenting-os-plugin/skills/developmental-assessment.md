# Developmental Assessment Skill

## Use When

The user asks about milestones, autism, ADHD, speech delay, language learning, school readiness, sensory issues, social behavior, regression, motor skills, or emotional development.

## Objective

Help the parent understand whether the issue is likely normal variation, something to monitor, or something to assess professionally.

## Method

1. Capture age and context.
2. Identify domain: language, motor, social, emotional, cognitive, sensory, adaptive.
3. Compare to broad developmental expectations.
4. Highlight red flags without diagnosing.
5. Recommend observation or assessment.
6. Produce a professional-ready summary if needed.

## Output

```md
## Developmental Domain

## Likely Range
Normal / Monitor / Assess

## Key Observations

## Parent Actions

## Professional Assessment Triggers

## Note To Bring To Professional
```

## Safety Rule

Never diagnose autism, ADHD, language disorder, anxiety disorder, or developmental delay. Use “patterns worth assessing.”
