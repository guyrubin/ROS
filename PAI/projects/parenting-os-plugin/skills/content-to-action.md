# Content To Action Skill

## Use When

The user asks to turn psychological theory, parenting content, expert material, videos, books, or professional knowledge into practical app content.

## Objective

Convert knowledge into parent actions, routines, scripts, checklists, and app modules.

## Method

1. Extract principle.
2. Translate into parent scenario.
3. Create action pattern.
4. Create script.
5. Create tracking measure.
6. Save as reusable module.

## Output

```md
## Principle

## Parent Scenario

## Action Pattern

## Parent Script

## App Module

## Tracking Signal
```

## Rule

No abstract theory unless it directly changes the parent’s next action.
