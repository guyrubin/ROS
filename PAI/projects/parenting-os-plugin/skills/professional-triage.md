# Professional Triage Skill

## Use When

The user needs to involve a pediatrician, psychologist, speech therapist, school, municipality, daycare, or family-support professional.

## Objective

Create clean, neutral, professional handoff notes.

## Output

```md
## Child Context

## Concern

## Observed Behaviors

## Frequency / Duration

## Triggers

## What Parents Tried

## What Helped

## What Did Not Help

## Questions For Professional

## Urgency Level
```

## Rules

* Use neutral language.
* Avoid diagnosis.
* Separate facts from interpretations.
* Make the note usable in clinical, school, or municipal settings.
